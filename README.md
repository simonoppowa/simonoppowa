### [🍴 OpenNutriTracker](https://github.com/simonoppowa/OpenNutriTracker)

OpenNutriTracker is a free and open-source calorie tracker built for simplicity, privacy, and a smooth cross-platform experience on **iOS** and **Android**.

**Highlights**
- Built with **Flutter** using a clean, scalable architecture
- Focused on a simple UX for tracking nutrition and daily habits
- Open-source and community-driven

**Tech Stack:** Dart, Flutter, BLoC, Provider, Hive, Supabase, Sentry, Material You, Barcode Scanning, Local Notifications, Android, iOS
